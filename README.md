# School Library

A comprehensive school library management system designed to streamline library operations and enhance user experience for students and staff.

## Features

- Book inventory management
- User registration and authentication
- Book borrowing and returning system
- Reservation management
- Search and catalog functionality
- Library statistics and reporting

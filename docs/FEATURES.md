# Features Specification - School Library System

## Phase 1: MVP (Minimum Viable Product)

### 1. User Management
- [x] User Registration (Students, Teachers, Librarians)
- [x] User Login with JWT Authentication
- [x] User Profile Management
- [x] Automatic Library Card Generation
- [x] Role-Based Access Control (Student, Teacher, Librarian, Admin)

### 2. Book Catalog Management
- [x] Add/Edit/Delete Books (Admin/Librarian only)
- [x] Book Information (Title, Author, ISBN, Genre, Publication Year)
- [x] Book Status Tracking
- [x] Book Location/Call Number Tracking
- [x] Multiple Copies Management
- [x] Search Books by Title, Author, ISBN

### 3. Borrowing System
- [x] Checkout Books
- [x] Return Books
- [x] Due Date Management
- [x] Overdue Tracking
- [x] Fine Calculation
- [x] View Borrowed Items
- [x] Renewal System

### 4. Basic Search & Discovery
- [x] Search by Title
- [x] Search by Author
- [x] Search by ISBN
- [x] Filter by Genre
- [x] Display Available Books
- [x] Pagination

### 5. Library Card System
- [x] Automatic Card Generation on Registration
- [x] Card Number Assignment
- [x] Card Expiry Dates
- [x] Card Status Tracking
